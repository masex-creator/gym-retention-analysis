# Gym Retention Analysis

This project analyzes gym member data to identify the key factors driving customer churn. The goal is to uncover patterns in member behavior and provide actionable recommendations to improve retention and long-term customer value.

## Business Problem

The goal of this analysis is to help gym management understand:

- Which members are most likely to churn  
- What factors contribute to churn  
- How to improve retention and long-term membership value  

## Dataset

The dataset includes:

- Age  
- Membership Type (Monthly, Quarterly, Yearly)  
- Visits Per Month  
- Churn Status (Yes/No)  

## Tools Used

- SQL (Google BigQuery)  
- Google Sheets / Excel  
- Tableau (for visualization)  

## Key Analysis Performed

- Churn rate by age group  
- Churn rate by membership type  
- Combined analysis (Age Group + Membership Type)  
- Average visits vs. churn behavior  

## Key Insights

- Members aged 40+ had the highest churn rate (~28.8%)  
- Quarterly memberships in the 40+ group had the highest churn overall (35%)  
- Monthly memberships showed consistently high churn across age groups (~29–31%)  
- Visit frequency differences were less significant than expected  

## Recommendations

- Incentivize members to switch from monthly to yearly plans  
- Introduce loyalty perks for the 40+ age group  
- Investigate why quarterly plans underperform (pricing, commitment, or perceived value)  
- Conduct surveys targeting high-risk churn groups  

## Conclusion

From this analysis, membership type stood out as the strongest driver of churn, more than age or visit frequency. Members on monthly and quarterly plans consistently had higher churn rates across all age groups.

Based on these findings, there is a clear opportunity to improve retention by encouraging members to move toward longer-term plans, which could lead to stronger engagement and greater long-term customer value.
